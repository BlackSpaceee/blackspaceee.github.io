# BlackSpace — 個人財務管理 Web App

一個平台追蹤收支、管理投資組合、設定預算目標。

## 功能

- **每月收支追蹤** — 按分類查看花費明細、預算 vs 實際對比
- **投資組合管理** — 即時股價、多幣別支援（USD/HKD/EUR/GBP/JPY）、自動換算 HKD
- **儲蓄目標** — 環形進度視覺化，追蹤達成率
- **AI 財務分析** — 基於真實數據的個人化建議
- **週期性自動記帳** — 設定每月固定支出，自動新增交易
- **智慧分類** — 輸入商戶名稱自動建議分類
- **資料匯出/匯入** — JSON 備份與還原
- **PWA 支援** — 可加入手機主畫面，像原生 App 使用
